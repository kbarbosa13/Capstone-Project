1. Clone the repo

git clone https://github.com/kbarbosa13/Capstone-Project.git

2. CD into the folder and follow the following instructions 

3. Install dependencies for the backend 

cd server
npm install

4. Install dependencies for the frontend

cd../frontend
npm install 

5. You can start the backend by doing 

cd server
node server.js

6. And you can start the frontend by opening up a new terminal and doing

cd frontend 
npm run dev
